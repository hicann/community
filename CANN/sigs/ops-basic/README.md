# ops-basic SIG
提供算子基础框架和调度能力，同时包括数学类算子、视觉类算子等。
本SIG负责opbase、ops-math、ops-cv仓，其在CANN架构中的位置如下图所示：
![alt text](image.png)

## 工作目标
- 负责基础框架和调度能力、数学类、视觉类算子领域技术路线探讨与规划。
- 组织基础框架和调度能力、数学类、视觉类算子领域例会，引导技术讨论和决策。
- 负责SIG范围内的代码仓的Bug、Issuse和用户诉求等处理。
