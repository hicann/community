# cann-recipes SIG
recipes领域相关的兴趣小组，负责cann平台样例仓建设，如LLM、多模态、空间智能、具身智能等算法领域的样例规划及建设

## 工作目标
- 1、负责recipes领域技术路线规划和落地，组织例会，引导技术讨论和决策；
- 2、负责周边组织沟通协调，推动落地相关项目；
- 3、负责社区Bug、issue和邮件列表等渠道反馈的问题分发处理。
