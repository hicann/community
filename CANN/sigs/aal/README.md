# AAL SIG

- 愿景：构建面向大语言模型、推荐、具身智能领域的专用加速库
- 负责领域加速库相关组件社区技术发展和决策
- 负责领域加速库相关软件包的规划、升级和维护
- 及时响应领域加速库用户反馈和解决领域加速库相关问题
