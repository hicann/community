# hccl SIG
提供集群通信库，开放底层基础通信能力，涵盖集合通信、点到点通信和单边通信等场景通信算子。

# 工作目标
- 负责通信基础能力和通信算子领域技术路线探讨与规划。
- 组织通信基础能力和通信算子领域例会，引导技术讨论和决策。
- 负责SIG范围内的代码仓的Bug、Issuse和用户诉求等处理。
