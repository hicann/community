# CANNBot SIG
CANN 社区 Agent 智能体兴趣小组，为社区提供AI辅助开发、算子生成、生态建设的智能体最佳实践。

SIG 更多介绍请通过 [CANNBot](https://gitcode.com/cann/cannbot-skills) 代码仓库进行了解。

## 工作目标
- 1、构建面向 CANN 社区的 Skill 能力，支撑不同编程范式下的算子生成、模型迁移等开发场景。
- 2、依托大模型搭建语料飞轮与自动评估体系，打造高质量算子生成语料库。
- 3、建立L0-L5算子生成等级评价体系，形成 CANN AI 生成能力度量标准，牵引模型迭代与算子开发落地。
- 4、打造 AI Agent 编码能力，提升开发效率，补齐长尾算子，持续提升 CANN 生态满足度。

## 职责与范围
   - 本 SIG 主要负责 [CANNBot](https://gitcode.com/cann/cannbot-skills) 代码仓库的开发与维护。
   - 定位：CANNBot 是面向 CANN 开发的系列智能体，旨在提升开发效率。
   - 功能：本 SIG 为其提供可复用的 skills 模块，目前已实现 Ascend C、PyPTO、TileLang 算子开发全流程覆盖，未来将拓展至 CANN 更多技术领域。
