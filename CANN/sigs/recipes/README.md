# recipes SIG

### 🌟 简介 (Introduction)

欢迎来到**recipes SIG**！我们致力于提供拿来即用的算法模型样例，帮助您简单、快速、高效地使用CANN进行模型开发及优化。

**已开源仓的列表：**

| 项目名称 | 算法领域 |
|----------|----------|
| [cann-recipes-infer](https://gitcode.com/cann/cann-recipes-infer) | LLM/多模态推理样例 |
| [cann-recipes-train](https://gitcode.com/cann/cann-recipes-train) | LLM/多模态训练样例 |
| [cann-recipes-spatial-intelligence](https://gitcode.com/cann/cann-recipes-spatial-intelligence) | 空间智能算法样例 |
| [cann-recipes-embodied-intelligence](https://gitcode.com/cann/cann-recipes-embodied-intelligence) | 具身智能算法样例 |
| [cann-recipes-harmony-infer](https://gitcode.com/cann/cann-recipes-harmony-infer) | 鸿蒙设备推理实践样例 |

### 🎯 愿景与使命 (Vision & Mission)

我们希望能够提供开发者最需要的指导：如何快速上手？如何复现业界SOTA模型？如何榨干NPU性能？

我们的使命是：**消除技术壁垒，沉淀优秀实践。**

我们希望通过提供高质量、可复用的算法样例，帮助开发者从“Hello World”平滑过渡到复杂模型开发优化，让CANN更好地帮助您的工作和科研。

### 🛠️ 我们主要关注 (Scope of Work)

我们专注于以下内容的建设与维护，每一个方向都需要您的参与：

*   **全场景算法样例**：覆盖大模型、多模态、空间智能、具身智能各领域的算法样例。
*   **高性能模型复现**：基于CANN深度优化的主流模型训练与推理脚本。
*   **特性优秀实践**：针对CANN新特性的使用指南与性能调优技巧。

### 🤝 加入我们 (Join Us)

在这里，您不仅仅是一个代码贡献者，更是CANN生态的建设者：

*   **技术成长**：直接与CANN优化专家交流，深入理解AI算法、软件、硬件协同优化原理。
*   **影响力**：您的代码将被成千上万的开发者使用，成为广泛的参考标准。
*   **荣誉激励**：优秀的贡献者将获得优秀贡献者认证，成为您简历上亮眼的一笔。

### 📬 联系方式 (Contact & Participation)

我们奉行“开放、透明、友善”的社区原则。无论您是提交一个Bug修复，还是新增样例，我们都诚挚欢迎。

在仓库列表中选一个感兴趣的仓库，根据仓内CONTRIBUTION.md中的介绍，开启您的CANN生态贡献之路吧！

我们欢迎您参与我们的SIG会议，在线交流，一起探讨

*   **[邮件订阅](https://mailweb.cann.osinfra.cn/mailman3/lists/recipes.cann.osinfra.cn/)**
*   **[在线会议](https://etherpad-cann.meeting.osinfra.cn/p/sig-recipes)**：每两周一次，周二上午10:30 - 12:00（UTC+8）
*   **飞书交流群**（扫码入群，若二维码失效请联系 Maintainer）
