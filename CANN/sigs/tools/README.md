# tools SIG

### 简介

欢迎来到**tools SIG**！我们致力于构建模型压缩和问题定位领域的完整技术生态，探索极致高效的模型优化路径和精准高效的问题诊断方法！

**工具列表**
|   名称   |  领域  |  介绍  |
|----------|----------|----------|
| [amct](https://gitcode.com/cann/amct) | 模型压缩 | 通过量化等优化算法，完成模型的“瘦身”和“提速”，提升模型部署性能 |
| [oam-tools](https://gitcode.com/cann/oam-tools) | 故障定位 | 为开发者提供故障定位和性能测试调优工具，提升问题定位效率 |

### 工作目标
- 1、负责工具领域技术路线规划和落地；
- 2、组织工具领域例会，引导技术讨论和决策；
- 3、负责周边组织沟通协调，推动落地相关项目；
- 4、负责SIG仓库权限和邮件列表等基础设施的管理；
- 5、负责社区Bug、issue和邮件列表等渠道反馈的问题分发处理。
