# ops-dxl SIG
作为大模型分布式集群和数据管理组件，提供了高性能、零拷贝的点对点数据传输、链路管理和缓存管理的能力，该能力通过简易的API开放给用户。

## 工作目标
- 负责大模型分布式集群和数据管理组件领域技术路线规划和落地；
- 组织大模型分布式集群和数据管理组件领域例会，引导技术讨论和决策；
- 负责周边组织沟通协调，推动落地相关项目；
- 负责SIG 仓库权限和邮件列表等基础设施的管理；
- 负责社区Bug、issue和邮件列表等渠道反馈的问题分发处理。
