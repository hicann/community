# render SIG

Render SIG 是聚焦渲染库的兴趣小组，致力于在昇腾AI处理器上实现3D GS、3D MESH等3D表征的高性能可微渲染处理。

# 工作目标

1. 负责可微渲染库领域（3D GS/3D MESH等）技术路线规划和落地。
2. 组织可微渲染库领域技术例会，引导技术讨论和决策。
3. 负责SIG范围内的代码仓Bug、Issuse和用户诉求等处理。
