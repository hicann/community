# crypto SIG

crypto SIG 是聚焦密码计算的兴趣小组，围绕昇腾 NPU 打造高性能密码软件库，作为 CANN 生态的密码计算公共能力层，致力于补齐"可调用、可验证、可优化"的 NPU 密码计算底座，支撑可信 AI 与数据安全。

# 工作目标

1. 功能目标：覆盖国密与国际主流算法，建设统一易用的密码算子与算法库。
2. 性能目标：面向高吞吐、低时延场景，发挥昇腾 NPU 并行计算能力，实现相对 CPU 基线的显著加速。
3. 生态目标：通过开源共建完善文档、测试与示例工程，沉淀密码算子开发规范与最佳实践。
4. 自主目标：面向国密合规、信创适配与数据安全场景，建设自主可控、可审计、可复用的密码计算能力。
