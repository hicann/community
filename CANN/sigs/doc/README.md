# doc SIG
管理CANN开源开放项目的文档开发工作，致力于为开发者提供具备竞争力的文档体验。

## 工作目标
- 负责CANN文档体系的探讨与规划。
- 组织CANN文档领域例会，引导技术讨论和决策。
